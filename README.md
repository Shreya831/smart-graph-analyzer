# 🚀 Smart Graph Analyzer

An interactive Graph Theory project that analyzes and visualizes complex networks.

## 🔍 Features
- 📊 Community Detection (Modularity-based)
- 🔥 Influencer Identification (Centrality)
- 🔗 Shortest Path Analysis
- 🌐 Interactive Graph Visualization (Plotly)
- ⚡ Built with NetworkX + Gradio

## 🛠 Tech Stack
- Python
- NetworkX
- Plotly
- Gradio

## 🎯 Use Cases
- Social Network Analysis
- Startup/User Network Insights
- Graph Theory Learning
- Data Science Projects

## 🚀 Demo
Run on Google Colab or deploy via Hugging Face Spaces.
